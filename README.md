# WebGL2 Renderer Demo

A quick project made in 2-3 days, with a WebGL2 renderer on canvas from scratch (and a linear algebra package), a demo with a Utah Teapot and a controllable camera

Built with Typescript and Webpack, tested on Firefox

## Camera controls

**Right click inside the red box to lock the pointer so that you're able to move the camera.**

- **`W`**: Move forwards
- **`A`**: Move right
- **`S`**: Move backwards
- **`D`**: Move left
- **`Space`**: Move up
- **`Left Shift`**: Move down

Use scroll wheel to adjust movement speed (up - faster, down - slower)
